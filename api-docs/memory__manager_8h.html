Detailed Description

Comprehensive memory monitoring and management system for ESP32-S3 dual display billboard.

This module provides real-time memory monitoring, health tracking, and automatic memory management for optimal system performance. Designed specifically for ESP32-S3 with PSRAM support and dual display configurations.

Key Features:

  • Real-time heap and PSRAM monitoring
  • Memory health scoring and alerts
  • Automatic memory cleanup triggers
  • Memory usage statistics and trending
  • Integration with system logger
