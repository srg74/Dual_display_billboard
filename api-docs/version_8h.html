Detailed Description

Version Information and Build Details for ESP32 Dual Display Billboard.

Version
0.9
Author
Dual Display Billboard Team
Date
2025-08-16

This header provides standardized access to version information automatically generated during the build process. Version information includes:

• Base version from platformio.ini (e.g., v0.9) • Build numbers in YYMMDDx format (e.g., 2508160) • Full version strings (e.g., v0.9-build.2508160) • Build timestamps and environment details

Usage:

#include "version.h"
LOG_INFO("System", "Firmware: " + String(getFirmwareVersion()));
LOG_INFO("System", "Build: " + String(getBuildNumber()));
