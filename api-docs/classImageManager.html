Detailed Description

Comprehensive image management system.

Handles all aspects of image storage and display for the billboard system. Integrates with DisplayManager for hardware control and provides web API endpoints for image upload and management.

Key features:

  • Automatic image count limiting (MAX_IMAGE_COUNT)
  • Size validation (MAX_IMAGE_SIZE)
  • Dimension validation per display type
  • LittleFS storage management
  • TJpg_Decoder integration for efficient rendering
  • Detailed error reporting
