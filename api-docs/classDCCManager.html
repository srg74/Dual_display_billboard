Detailed Description

Digital Command Control accessory decoder for model railway integration.

Implements a DCC accessory decoder that responds to turnout commands to control display modes. Provides seamless integration between model railway DCC systems and the dual display billboard functionality.

The manager operates as a standard DCC accessory decoder, listening for commands on a configurable address. When commands are received, it automatically switches the display system between gallery slideshow mode and clock display mode.

Technical Implementation:

  • Uses NmraDcc library for robust DCC signal decoding
  • Configurable GPIO pin with internal pull-up for reliable signal reception
  • Static callback integration to meet NmraDcc C library requirements
  • Address validation ensures commands target the correct decoder
  • State management with command timestamp tracking
